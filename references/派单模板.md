# Stage 2 并行 Agent 派单模板

Stage 2 同一条消息里并行发 ≥ 3 个 Agent。每个 prompt 都要带齐：主题、Angle、用户基础与目标程度（决定资料深浅）、资料语言偏好、搜索词、必覆盖问题、Return format。下面是三路默认模板，按主题替换 `[主题]`。

---

## Agent 1 · 系统资料 / 知识体系

```
你在为「[主题]」的学习包做资料采集。学习者基础：[基础]，目标程度：[目标]，资料语言：[中英文偏好]。

任务：联网搜索，搞清这个主题"由哪些知识模块组成、从哪学到哪、有哪些必懂概念"。

必覆盖：
- 这个主题的权威入门资料、官方文档或官方教程（如果有）
- 公认的经典书目 / 高质量长文 / 优质博客
- 核心概念清单（每个概念一句话能说清的程度）
- 合理的知识模块拆分（这个主题可以拆成哪几块来学）

要求：
- 每条资料给真实标题 + 真实可访问 URL + 类型(官方/书/教程/长文) + 一句话为什么推荐
- 资料具体到名字，查不到权威资料就如实说"暂未找到"，绝不编书名/链接
- 英文资料保留原标题，补一句中文说明

Return format：按"知识模块"分组，每组列资料(标题+URL+类型+推荐理由) + 该模块核心概念。
```

---

## Agent 2 · 优质视频 / 创作者

> 派这个 Agent 前，必须已拿到用户的「视频平台偏好」（优先 YouTube / YT+B站都要 / 只要 B站），写进下面 prompt 的搜索范围。

```
你在为「[主题]」的学习包采集优质视频。学习者基础：[基础]，目标程度：[目标]。

视频平台偏好（用户已选）：[优先 YouTube / YouTube 和 Bilibili 都要 / 只要 Bilibili]。
  · 优先 YouTube → 主搜 YouTube，B站 仅在 YouTube 缺位时补充
  · YouTube 和 Bilibili 都要 → 两平台都认真覆盖
  · 只要 Bilibili → 只搜 B站，不收 YouTube
（小红书/抖音此类技术主题内容稀缺，只在用户特别要求时才找。）

任务：按上面平台偏好，找讲这个主题最好的视频和博主。

每条视频必须给到：
- 视频标题 + 真实可点链接
- 博主名 + 博主粉丝量/订阅量（真实数据，查不到写"未获取"）
- 该视频互动量：播放/点赞/评论（有多少填多少，查不到写"未获取"）
- 一句"为什么值得看 / 讲得好在哪"
- 这条适合学习的哪个阶段（入门 / 上手 / 进阶）

要求：
- 链接必须真实，绝不编造视频或频道
- 粉丝量、互动量是真实数据，没查到就如实标"未获取"，不许估算编造
- 优先口碑好、讲解清楚、更新不太过时的

Return format：按"入门 / 上手 / 进阶"分组列视频；末尾附"值得长期关注的博主"小表(博主+平台+粉丝量+擅长方向+主页链接)。
```

---

## Agent 3 · 学习路径 / 过来人经验

```
你在为「[主题]」的学习包采集"怎么入门、学习路线、避坑"的真实经验。学习者基础：[基础]，目标程度：[目标]，每天能投入：[时间]。

任务：在 Reddit / 知乎 / V2EX / 即刻 / 垂直论坛 等社区，找过来人怎么规划学这个主题。

必覆盖：
- 真实的阶段划分（先学什么、再学什么、最后学什么）
- 新手最常卡住、最容易放弃的点
- 不值得花时间的弯路、过时的内容、不用买的东西
- 怎么验证"我这一阶段学会了"
- 如果存在两种主流路线（社区有分歧），都要带回来，标清各自适合谁

要求：
- 多人反复提到的才算"公认路线"，单个匿名帖只能当参考，要标明
- 给出处链接，绝不编造经验帖

Return format：阶段化路线建议 + 常见卡点 + 避坑清单 + 验证方式 + （如有）两种路线对比，各带来源链接。
```

---

## Agent 4（可选）· 实战项目 / 练手清单

```
为「[主题]」找适合上手的小项目 / 练习 / 数据集 / 工具，按由易到难排，每个说明练什么、在哪做、产出是什么，带真实链接。
```

## Agent 5（可选）· 用户特别在意的角度

```
按用户特别要求（如只要免费资源 / 只要中文 / 只想最快接活）做定向补充采集，规则同上：真实来源、不编造。
```

---

## 派单注意

- 用户基础和目标程度要真传进每个 Agent：零基础 + 入门玩玩 → 资料偏轻、偏中文、偏视频；有基础 + 系统精通 → 可纳入一手英文文档、论文、源码。
- 三路是下限，命中"我只想要免费/只要中文/要练手项目"等明确诉求时加派 Agent 4/5。
- 所有 Agent 统一底线：真实链接、真实数据、查不到如实标注、绝不编造。
